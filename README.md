<h1> 🎈 Making Thumbnail image (블로그 썸네일 만들기) 🎈 </h1>
<ul>
<li>블로그 작성 시 썸네일을 매번 만들어내는 게 번거로워서 만들어낸 썸네일 자동 프로그램입니다.</li>
<li>폰트: NotoSansKR(Bold) 
  <br>=> 코드에서 자유롭게 폰트 변경 가능 :  font = ImageFont.truetype('폰트파일 전체경로 적기', size=50)</li>
<li>배경색상 : 랜덤변경</li>
<li>글자수 : 최대 33자</li>
</ul>
